Flexbox - torna seus itens responsivos

Flex-direction: Uma propriedade que estabelece um
eixo principal do container, define a direção que os
flex itens são colocados no flex container.

EIXOS

Row - direção do texto, esquerda para direita
- Row reverse - sentido oposto da direção padrão
- Column - ordena de cima para baixo, em uma unica coluna
- Column reverse - ordem inversa, de baixo para cima

Flex Wrap - Define se os itens devem ou não quebrar a
linha. 
- Nowrap - é o padrçao, não permite a quebra de linha
- Wrap - permite a quebra de linha assim que um dos flex
itens não puder mais ser compactado.
- Wrap-reverse - permite a quebra de linha no sentido 
contrario, mantém na primeira linha.
